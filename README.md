为什么不写博客？

感觉博客是不受控制的，假如有天博客不再支撑了，需要花时间去迁移和整理，否则以前的心血就流失了。用Latex完全可以本地化，而且写好就是一本还算正规的书，随时可以打印出来细细查看。Latex写出来文字漂亮、格式美观。平时的时间非常碎片化，知识不完备，不容易写出优质内容，用Latex工作中可以随时翻看。

为什么托管到GitHub？

其实GitHub也不知道哪天就不能访问，本地也可能丢失，算是一个“容灾”策略。顺便用用Git提升逼格。
